# Portfolio website

[![Deployed on Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black?style=for-the-badge&logo=vercel)](https://vercel.com/gngawe03-2686s-projects/v0-portfolio-website)

## Overview

Personal portfolio built with Next.js and Tailwind CSS.

## Build your app

To run locally:

```bash
pnpm install
pnpm dev
```

## How It Works

1. Next.js app using the App Router in `app/`
2. Tailwind CSS v4 for styling with CSS variables for theming
3. Deployed to Vercel from this repository
